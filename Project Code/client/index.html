Basic Chess Project

Play a complete chess game in the browser with save and replay support.

This first version focuses on the essentials: a playable board, legal move validation, optional AI moves, and browser-based game history without MongoDB or Docker.

Live Play

Move pieces directly on the board, switch modes, and play against the built-in AI.

Saved Games

Completed matches can be stored in local browser storage with full move history.

Replay Viewer

Load any saved match and step through the game move by move from the opening onward.
